# Proyecto Servidores Web

## Descripción
Aplicación web completa: backend en Node.js + MongoDB, frontend estático servido con Nginx, contenedores Docker.

Funcionalidad:
- Crear y unirse a salas
- Votación de restaurantes
- Interfaz básica con HTML, CSS y JS
- Persistencia de datos en MongoDB
- Autenticación con JWT

## Tecnologías
- Backend: Node.js, Express, Mongoose
- Base de datos: MongoDB
- Frontend: HTML, CSS, JS
- Contenedores: Docker + Docker Compose

## Instalación
1. Instalar Docker Desktop y Docker Compose
2. Clonar repositorio
3. Configurar archivo `.env` en `backend/`
4. Ejecutar:
   docker-compose up --build
